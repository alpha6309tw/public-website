# 常用 Markdown 語法說明

Markdown 是一種輕量級標記語言，讓我們可以用易讀易寫的純文字格式編寫文件，然後轉換成有效的 XHTML（或者 HTML）。

## 標題 (Headers)
使用 `#` 符號來表示標題，`#` 的數量代表標題的層級。

```markdown
# 第一層標題 (H1)
## 第二層標題 (H2)
### 第三層標題 (H3)
#### 第四層標題 (H4)
```

## 粗體與斜體 (Bold and Italic)
使用星號 `*` 或底線 `_` 來強調文字。

```markdown
**粗體文字**
*斜體文字*
***粗斜體文字***
```

## 清單 (Lists)

### 無序清單 (項目符號)
使用 `-`、`+` 或 `*` 加上一個空格來建立清單。

```markdown
- 項目一
- 項目二
  - 子項目 A
  - 子項目 B
```

### 有序清單 (編號)
使用數字加上 `.` 來建立清單。

```markdown
1. 第一點
2. 第二點
3. 第三點
```

## 連結 (Links)
連結的語法是 `[顯示文字](網址)`。

```markdown
[Google 首頁](https://www.google.com)
```

## 圖片 (Images)
圖片的語法跟連結很像，只是前面多了一個驚嘆號 `!`。
語法：`![圖片替代文字](圖片網址)`

```markdown
![Logo圖片](/assets/images/logo.png)
```

## 引用 (Blockquotes)
使用 `>` 符號來表示引用文字。

```markdown
> 這是一段引用文字。
> 這是第二行引用。
```

## 分隔線 (Horizontal Rules)
使用三個或更多的 `-`、`*` 或 `_` 來建立分隔線。

```markdown
---
```
